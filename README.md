# 🏦 Banking & Fraud Detection System (MSSQL)

## 📌 Project Overview

This project simulates a real-world banking system with transaction management and fraud detection using Microsoft SQL Server.

---

## 🎯 Problem Statement

Banks need to:

* Manage customer accounts
* Handle secure transactions
* Detect suspicious activities in real-time

---

## 🧱 Database Design

Tables used:

* Customers
* Accounts
* Transactions

---

## 🔁 Business Logic

* Transactions use COMMIT / ROLLBACK
* Money transfer is atomic (safe)
* Fraud detection is automated using triggers

---

## ⚙️ Advanced SQL Features

* Transactions (ACID properties)
* Triggers (real-time fraud detection)
* Stored Procedures
* Window Functions
* CASE statements

---

## 🚨 Fraud Detection Rules

* Large transactions (> 50,000)
* Frequent transactions in short time
* Abnormal transaction patterns

---

## 📊 Key Queries

* Suspicious transaction detection
* Account activity analysis
* Risk scoring
